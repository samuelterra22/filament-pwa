# Changelog

All notable changes to `filament-pwa` will be documented in this file.

## v1.0.0

- Complete rewrite following FilamentPHP plugin-skeleton standards
- New namespace: `SamuelTerra22\FilamentPwa`
- Uses `spatie/laravel-package-tools` for service provider
- Supports Filament v4
- 49 languages supported
- Dark mode support (inherits Filament panel theme)
- Fixed translations system
- Improved Settings Page with tabbed UI
- Better service worker with network-first strategy for navigation
- Select dropdowns for Display Mode and Orientation
- Cleaner property names (removed `pwa_` prefix)
- Full CI/CD with GitHub Actions (tests, PHPStan, Laravel Pint)
